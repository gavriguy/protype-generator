# Protype Generator

A Yeamon Generator for creating HTML React based prototypes.

## Usage

Install `yo` `generator-protype`

`npm install -g yo generator-protype`

Make a new directory, and cd into it:

`mkdir my-new-project && cd $_`

Run yo protype

`yo protype`
